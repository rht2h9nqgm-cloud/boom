# 💥 Boom - Outlook Email Manager

A complete, production-ready Outlook email management system built with React + Vite + Tailwind CSS.

## ✨ Features

- 📧 **Email Management**: Read, send, delete emails
- 🔍 **Advanced Search**: Search by subject, sender, content
- 📁 **Multi-folder Support**: Inbox, Sent, Drafts, Junk, Deleted
- 🔐 **OAuth Authentication**: Device code + Real login
- 📱 **Responsive UI**: Works on desktop & mobile
- ⚡ **Fast Performance**: Vite + React optimization
- 🎨 **Modern Design**: Tailwind CSS styling
- 🌐 **API Integration**: Cloudflare Workers backend

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
git clone https://github.com/YOUR_USERNAME/boom.git
cd boom
npm install
npm run dev
